# 특허시세 (PATENT-SISE)

> 등록특허번호 하나로 IP담보대출·보증 적격성을 3분 만에 사전진단하는 "특허판 KB부동산 시세".
> 2026 지식재산 데이터 활용 창업 경진대회 출품 프로토타입.

공식 특허가치평가(통상 4~6주·약 500만원)를 대체하지 않는 **참고용 사전진단 깔때기**입니다.
번호를 입력하면 ①담보 적격성 관문 진단 ②정량지표 13종 ③4축 점수+담보적합 등급 A~E
④XAI 여신심사 메모(Claude)를 한 번에 제공합니다.

## 기술 스택
- Next.js 16 (App Router) + TypeScript + Tailwind CSS v4
- **webpack 잠금**: `next dev --webpack` / `next build --webpack` (상위 한글 경로에서 Turbopack panic 회피)
- Claude 통합: `@anthropic-ai/sdk` 스트리밍, 모델 `claude-haiku-4-5-20251001`
- 외부 차트/파서 의존성 0 (순수 SVG 레이더 + 경량 XML 파서)

## 실행
```bash
npm install
cp .env.example .env.local   # 값 입력 (ANTHROPIC_API_KEY, DATA_GO_KR_SERVICE_KEY)
npm run dev                  # http://localhost:3000
npm run build                # 프로덕션 빌드(webpack)
```

### 키 미등록 / 폴백 / 캐시
- 현재 계정 키는 KIPRIS 활용신청 전 단계 → 호출 시 `resultCode 30`(SERVICE KEY IS NOT REGISTERED).
- 이 경우 `data/samples/` 의 데모 샘플 3건으로 자동 폴백하고 UI에 **"데모 샘플 데이터"** 배지 표시.
- 실호출 성공 시 `.cache/` 파일 캐시에 저장(월 1,000회 쿼터 방어).
- 활용신청 완료 후 **코드 수정 없이** 실데이터로 전환(`lib/kipris.ts` ENDPOINTS 상수만 검증).

|---|---|---|
| 1 법적상태 | 등록사항(15058125) | 금융적합성 |
| 2 잔존 존속기간 | 특허서지(15058788) | 활용성 |

| 7 무효심판 건수·생존 | 심판사항(15065474) | 권리성 |
| 8 권리범위확인심판 | 심판사항(15065474) | 권리성 |
| 9 후방인용 문헌 수 | 심사인용문헌(15057617) | 기술성 |
| 10 출원→등록 소요기간 | 특허서지(15058788) | — |
| 11 우선권/패밀리 | 특허서지(15058788) | — |
| 12 IPC 분류·기술분야 | 특허서지(15058788) | 기술성 |
| 13 **전방인용(피인용) 수** | **피인용문헌(15002128)** | 기술성 |

- 결측은 반드시 **N/A + 신뢰도 표시**, 추정값 생성 금지(감사가능성 핵심).
- 점수·등급은 코드로 직산(`lib/score.ts`). 생성형 AI는 관여하지 않으며, 메모(`/api/memo`)에서 근거 서술만 담당.
- 담보 적격성 관문: 법적상태 유지 · 단독 권리 · 심판 계류 없음. 미통과 시 등급 상한 제한.

## 심사위원 데모 진입 경로 (재현 가능)
1. 홈(`/`) 접속 → 히어로 아래 **"샘플 특허로 바로 체험"** 섹션의 3개 카드(프리셋 버튼) 중 하나 클릭:
   - **이차전지 분리막** (`1020210012345`) — 단독 권리·무심판 → 관문 통과, 등급 A
   - **AI 진단보조 SW** (`1020190098765`) — 공유 특허·권리이전 2회 → 관문 미통과(공유)
   - **친환경 포장소재** (`1020170054321`) — 무효심판 계류·피인용 N/A → 관문 미통과(무효)
2. 또는 히어로 입력창에 번호 직접 입력 → "3분 사전진단" → `/report?no={번호}`.
3. 리포트에서 관문 배너·등급·레이더·13지표 카드·Claude 메모 스트리밍 확인. "🖨 은행 제출용 1페이지 인쇄" 버튼으로 PDF 저장(print CSS).
